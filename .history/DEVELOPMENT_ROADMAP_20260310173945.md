# 🚀 MAC-ADG 系统开发路线图与功能清单

## 项目现状总结 ✅

已完成的核心模块：

- ✅ **数据库层** (database/): Models、Connection、初始化逻辑
- ✅**Scout Agent** (scout_agent.py): Crossref API 元数据获取
- ✅ **Vision Agent** (vision_agent.py): 浏览器截图、视觉分析
- ✅ **Judge Agent** (judge_agent.py): 身份匹配、标记识别 (* #)
- ✅ **Orchestrator** (orchestrator.py): 流程编排、批量处理
- ✅ **前端页面** (frontend/pages + pages/): Streamlit UI 框架
- ✅ **工具库** (utils/): PDF 加载器、Excel 解析器
